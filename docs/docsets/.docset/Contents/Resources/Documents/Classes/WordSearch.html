class WordSearch

Word Search Object Model to draw words in 1-D and 2-D Arrays as Labels

Use by:

let wordSearch = WordSearch(usingGridSize: 10, withDifficulty: .hard)
wordSearch.getDefaultLabels()
wordSearch.make()
let labels = wordSearch.collapsedLabels()
